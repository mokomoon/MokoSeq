# Workspace System

MokoSeq uses a simple, clean folder structure to organize your data.

## Root directory

data/

## Workspace level 

data/<workspace>/

## Project level

data/<workspace>/<project>/

## Files

data/<workspace>/<project>/files/

## Categories
Stored in:

data/<workspace>/<project>/categories.json


This structure makes it easy to:
- back up your work  
- move projects  
- inspect files manually  
- integrate with other tools
