# Changelog

All notable changes to this project will be documented here.  
This project follows a simple, human‑readable format — no strict versioning rules yet.

---

## [0.1.0] — Initial Release
### Added
- Workspace + project system  
- File upload support (FASTA / FASTQ)  
- Category assignment with automatic saving  
- Obsidian‑style sidebar file tree  
- Mobile‑friendly collapsible sidebar  
- Sequence preview (first 500 characters)  
- Modular Python package (`mokoseq/`)  
- Basic documentation structure (`docs/`)  
- Placeholder tests (`tests/`)  
- Clean project layout with `pyproject.toml` and `setup.py`  

### Notes
This is the first public version of MokoSeq.  
The foundation is in place — future updates will focus on analysis tools, UI improvements, and expanded documentation.
