# Usage Guide

## 1. Create or select a workspace
Workspaces help you separate different projects or datasets.

## 2. Create or select a project
Each project contains:
- uploaded files  
- categories  
- analysis previews  

## 3. Upload files
Supported formats:
- FASTA  
- FASTQ  

Files are stored under:

data/<workspace>/<project>/files


## 4. Assign categories
You can group files into categories (e.g., "Mitochondrial", "Bacterial", "QC Passed").

Categories are saved automatically.

## 5. Preview sequences
Click a file tab to see:
- first 500 characters  
- placeholder analysis
